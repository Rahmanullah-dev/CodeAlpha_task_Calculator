# 🧮 Professional Calculator

A modern and responsive calculator built using **HTML5**, **CSS3**, and **Vanilla JavaScript**. This project performs basic arithmetic operations with a clean user interface, keyboard support, and mobile-friendly design.

## ✨ Features

* Basic arithmetic operations

  * Addition (+)
  * Subtraction (-)
  * Multiplication (×)
  * Division (÷)

* Responsive design for desktop and mobile devices

* Modern glassmorphism-inspired interface

* Keyboard support

* Real-time expression preview

* Delete last digit functionality

* Clear all (AC) functionality

* Decimal number support

* Division-by-zero protection

* Input length protection

* Clean and user-friendly UI

## ⌨️ Keyboard Shortcuts

| Key       | Action            |
| --------- | ----------------- |
| 0-9       | Enter Number      |
| .         | Decimal Point     |
| +         | Addition          |
| -         | Subtraction       |
| *         | Multiplication    |
| /         | Division          |
| Enter     | Calculate Result  |
| Backspace | Delete Last Digit |
| Escape    | Clear Calculator  |
| Delete    | Clear Calculator  |
